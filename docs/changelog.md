# 改动记录

## 0.4.1

- 新增会话分叉：可从任意已完成消息分叉出新会话，原会话保持不变。
- 分叉会话顶部显示"分叉自 xxx 第 N 条"来源横幅，点击跳回原会话并定位分叉点；原会话删除后降级为纯文本快照。
- 编辑消息时新增「分叉后发送」选项，与原地「重新发送」并列，保护原会话不被截断。
- Responses 模式：`previous_response_id` 失效或过期时自动回退为全量历史重发一次，老会话隔久续聊不再直接报错。
- Markdown / JSON 导出携带分叉来源信息。
- 性能优化：Markdown 渲染按需加载减小首屏体积，会话持久化引入缓存减少全量读写。

## 0.4.0

- 增加 Responses API 模式，可使用 `previous_response_id` 传递上下文。
- Responses 模式接入 `web_search` 内置工具，并在回复中展示工具调用状态。
- 设置页增加 API 模式和联网搜索开关。
- 支持连接诊断，帮助识别认证、地域 / 模型、网络和超时问题。
- 增加会话导出、usage 汇总、消息编辑重发、重新生成和错误重试等日常操作。

## 0.3.x

- 完善多会话管理：搜索、置顶、归档、删除和重命名。
- 修复归档会话默认选中、Responses 上下文混用、设置持久化等问题。
- 优化本地会话读取和渲染侧性能。

## MVP

- 建立 Electron + React + TypeScript 桌面应用骨架。
- 接入 DashScope / 阿里云百炼 OpenAI 兼容 Chat Completions 流式聊天。
